1. How many games are there?

SELECT count(*)
FROM game

2. List the Title and Developers of all games ordered by units sold from large to small.

SELECT Title, Developers
FROM game
order by Units_sold_Millions desc

3. What is the average units sold in millions of the games that are not developed by Nintendo?

SELECT avg(Units_sold_Millions)
FROM game
where developers != 'Nintendo'

4. What are the names and market districts of all platforms?

SELECT Platform_name, Market_district
FROM platform

5. What are the names and id of platforms whose download rank is 1?

SELECT Platform_name, id
FROM platform
WHERE Download_rank = 1

6. What are the maximum and minimum rank of the year of players.

SELECT max(Rank_of_the_year), min(Rank_of_the_year)
FROM player

7. How many players have rank of the year smaller than 3?

SELECT count(*)
FROM player
where Rank_of_the_year <= 3

8. List all player names in ascending alphabetical order.

SELECT Player_name
FROM player

9. List names and colleges of all players in descending order of rank of the year.

SELECT Player_name, College
FROM player

10. Please show the names and rank of players that have played the game titled "Super Mario World".

SELECT T3.Player_name, T3.rank_of_the_year
FROM game as T1
     join game_player as T2 on T1.Game_ID = T2.Game_ID
     join player as T3 on T2.Player_ID = T3.Player_ID
Where T1.Title = "Super Mario World"

11. Show the distinct developer of games played by players that go to college "Auburn".

SELECT distinct T1.Developers
FROM game as T1 join game_player as T2 
on T1.Game_ID = T2.Game_ID
join player as T3
on T2.Player_ID = T3.Player_ID
Where T3.College = "Auburn"

12. What is the average number of units sold in millions of games played by players with 
position "Guard"?

SELECT avg(Units_sold_Millions)
FROM game as T1 join game_player as T2 
on T1.Game_ID = T2.Game_ID
join player as T3
on T2.Player_ID = T3.Player_ID
Where T3.Position = "Guard"

13. Please list the title and platform name of games.

SELECT T1.Title, T2.Platform_name
FROM game as T1 join platform as T2 
on T1.Platform_ID = T2.Platform_ID

14. Please list the title of games with platforms that have market district in Asia
or USA.

SELECT T1.Title
FROM game as T1 join platform as T2 
on T1.Platform_ID = T2.Platform_ID
Where T2.Market_district = "Asia" or T2.Market_district = "USA"
