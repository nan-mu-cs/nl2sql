Find the name and hours of project that has the most number of scientists.

Select T1.name
From projects as T1 join assignedto as T2 on T1.code = T2.project
Group by T2.project
Order by count(*) desc
Limit 1

Find the name of the project which scientist whose name contains word ‘Smith’ is assigned to.

Select T2.name
From assignedto as T1
     Join projects as T2 on T1.project = T2.code
     Join scientists as T3 on T1.scientist = T3.SSN
Where T3.name LIKE '%Smith%'

Find the total hours of the projects which scientists Michael Rogers or Carol Smith is working on.

Select sum(T2.hours)
From assignedto as T1
     Join projects as T2 on T1.project = T2.code
     Join scientists as T3 on T1.scientist = T3.SSN
Where T3.name = 'Michael Rogers' or T3.name = 'Carol Smith'

List all the scientists' names, their projects' names, and the hours worked by that scientist on each project, in alphabetical order of project name, then scientist name.

SELECT T1.Name, T3.Name, T3.Hours
FROM Scientists as T1
     JOIN AssignedTo as T2 ON T1.SSN=T2.Scientist
     JOIN Projects as T3 ON T2.Project=T3.Code
ORDER BY T3.Name, T1.Name
