1. How many climbers are there?

FROM climber

2. List the names of buildings in descending order of points.

SELECT Name
FROM climber

3. List the names of climbers whose country is not Switzerland. 

SELECT Name
FROM climber
Where Country != "Switzerland"

4. What is the maximum point for players whose country is United Kingdom?

SELECT max(Points)
FROM climber
Where Country = "United Kingdom"

5. How many distinct countries are the climbers from?

SELECT T2.Height
FROM climber as T1 join mountain as T2 
ON T1.Mountain_ID = T2.Mountain_ID
order by T1.Points DESC limit 1

13. Show the distinct names of mountains climbed by climbers from country "West Germany".

SELECT distinct T2.Name
FROM climber as T1 join mountain as T2 
ON T1.Mountain_ID = T2.Mountain_ID
Where T1.Country = "West Germany"
